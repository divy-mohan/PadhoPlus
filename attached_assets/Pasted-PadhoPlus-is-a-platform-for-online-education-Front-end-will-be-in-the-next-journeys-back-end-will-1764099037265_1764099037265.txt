1. User roles & main journeys
Roles
•	Student (primary)
•	Parent (optional, can track performance)
•	Teacher / Mentor (you + future educators)
•	Admin (you / core team)
Core journeys for a student
1.	Land on Home → Explore batches/courses → View batch detail → Watch demo → Enroll (free/paid) → Start learning.
2.	Existing student: Login → Dashboard → Continue watching → Practice/Tests → Track progress & ranks.
3.	Doubt flow: From lecture → Ask doubt (image/text/audio) → Mentor answers → Student gets notification.
We’ll design the website structure around these.
